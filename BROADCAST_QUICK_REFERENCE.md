# Быстрая справка по рассылкам

## Форматы фильтров

| Тип     | Формат                          | Пример                              |
| ------- | ------------------------------- | ----------------------------------- |
| Все     | `все`                           | `все`                               |
| Регион  | `регион: <название>`            | `регион: Москва`                    |
| Вид ТСР | `тср: <вид>`                    | `тср: коляска`                      |
| Способ  | `способ: <тип>`                 | `способ: сертификат`                |
| Комбо   | `комбо: параметр=значение, ...` | `комбо: регион=Москва, тср=коляска` |

## Переменные для персонализации

| Переменная     | Что подставляется   | По умолчанию                   |
| -------------- | ------------------- | ------------------------------ |
| `{{имя}}`      | Имя пользователя    | username или "пользователь"    |
| `{{регион}}`   | Регион пользователя | "ваш регион"                   |
| `{{тср}}`      | Виды ТСР            | "назначенные ТСР"              |
| `{{дата_тср}}` | Дата получения ТСР  | "следующая дата получения ТСР" |

## Шаблоны сообщений

### Напоминание о ТСР

```
Здравствуйте, {{имя}}!

Напоминаем: {{дата_тср}} - день получения ваших ТСР.

Виды ТСР: {{тср}}

Подготовьте документы заранее.
```

### Региональное объявление

```
{{имя}}, важная информация для жителей {{регион}}!

[Ваш текст объявления]
```

### Информирование по типу ТСР

```
Уважаемый(ая) {{имя}}!

Специально для получателей: {{тср}}

[Ваш текст]
```

## Примеры комбинаций

```
комбо: регион=Москва, тср=коляска, способ=сертификат
комбо: регион=Санкт-Петербург, способ=выдача
комбо: тср=протезы, способ=сертификат
```

## Частые вопросы

**В: Можно ли отменить рассылку?**
О: Да, напишите "отмена" на любом этапе

**В: Что если пользователь не заполнил профиль?**
О: Будут использованы значения по умолчанию

**В: Как проверить, сколько людей попадет в рассылку?**
О: Система покажет количество перед отправкой

**В: Можно ли отправить фото?**
О: Да, можно отправить одно или несколько фото

**В: Как работает фильтр по ТСР?**
О: Ищет указанное слово в списке ТСР пользователя (например, "коляска" найдет "инвалидная коляска")
